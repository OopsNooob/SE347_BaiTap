Giải phương trình bậc 1 và bậc 2

ax + b = 0
ax² + bx + c = 0

📚 Công Thức:

Bậc 1 (ax + b = 0):

• Nếu a ≠ 0: x = -b/a

• Nếu a = 0 và b ≠ 0: Vô nghiệm

• Nếu a = 0 và b = 0: Vô số nghiệm


Bậc 2 (ax² + bx + c = 0):

• Δ = b² - 4ac

• Δ > 0: Hai nghiệm phân biệt

• Δ = 0: Nghiệm kép

• Δ < 0: Vô nghiệm
